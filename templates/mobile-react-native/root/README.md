# React Native Mobile Bootstrap

This template scaffolds a React Native (Expo) app for the AI fund launch flow.

## Stack

- React Native
- Expo
- TypeScript

## Run

```bash
npm install
npm run start
```

## User Flow Included

1. Intake form (risk level, portfolio holdings, amount, horizon)
2. Portfolio dashboard with 4 animated AI team windows
3. Full-view agent panel with voice-demand control UI
